# Sales Performance & Profitability Analysis Dashboard

## Project Overview

This project presents an interactive Power BI dashboard designed to analyze sales performance, profitability, customer behavior, and regional sales trends. The dashboard provides key business insights through visualizations and KPIs, enabling users to monitor sales performance and make data-driven decisions.

## Dashboard Features

### Key Performance Indicators (KPIs)

* Total Sales
* Average Sale Value
* Total Profit
* Total Quantity Sold

### Interactive Visualizations

* Sales Trend Analysis
* Sales by Region
* Product Category Performance
* Customer Type Distribution
* Profit by Product Category
* Sales by Sales Representative

### Filters and Slicers

* Date Range
* Region
* Customer Type
* Product Category

## Tools Used

* Power BI
* Power Query
* DAX (Data Analysis Expressions)

## Business Insights

* Identifies top-performing regions.
* Analyzes sales trends over time.
* Evaluates profitability across product categories.
* Compares customer segments.
* Tracks sales representative performance.
